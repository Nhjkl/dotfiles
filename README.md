# dotfiles
my dotfiles

# 使用 stow 管理

> mac install `brew install stow`

> linux install `yay -S stow`

# 使用

```bash
cd dotfiles

stow macprofile
```

install zsh settings for the root user

`sudo stow zsh -t /root`
